💡 Editor Help

This editor supports standard Markdown formatting with live preview on the right.

Keyboard Shortcuts

  • Ctrl+B → Bold — wraps selection in **...**
  • Ctrl+I → Italic — wraps selection in *...*
  • Ctrl+L → Insert link — wraps selection as [text](url)
  • Ctrl+Z / Ctrl+Y → Undo / Redo
  • Ctrl+F → Search inside the editor
  • Tab → Confirm autocomplete suggestion or jump to next link placeholder

Markdown Basics

  • ## Heading → Section heading
  • **bold**, *italic*
  • - item → Bullet list
  • [text](url) → Link
  • ![alt](url) → Embedded image
  • > text → Blockquote
  • --- → Horizontal divider

Quick Tags

Type [ followed by letters to autocomplete game terms. Use Enter or Tab to confirm.

  • [fg] → Factions Game
  • [gm] → Global Map
  • [faction], [character], [place], and more

Link Autocomplete

Type [ to also search internal pages by name (e.g. [Castle, [Eon 1024). Selecting a suggestion inserts a full markdown link.

Submitting Edits

Write an optional edit message in the text field at the bottom, then click the send button (→) to submit. Your edit will be queued for review. Edits submitted within one minute replace one another.
