Editor Help

This editor supports standard Markdown format.

Keyboard Shortcuts

  • Ctrl+B → Bold
  • Ctrl+I → Italic
  • Ctrl+L → Insert [Link]()

Quick Tags

Type [ followed by letters to autocomplete game terms. Use Enter or Tab to confirm.

  • [fg] → Factions Game
  • [gm] → Global Map
  • [faction], [character], [place], etc.

Link Autocomplete

Type [ to also find internal pages (e.g. [Castle, [Eon 1024).
